# napp-boilerplate
WIP boilerplate for node web applications.

## Install

```sh
$ npm install
```

## Usage

Run Express server:

```sh
$ npm start
```

Build JS applications:

```sh
$ npm run build
$ npm run build.prd (files are concatenated & minified)
```
