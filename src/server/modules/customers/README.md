# napp-boilerplate-customers-module
Customer Module

## Install

```sh
$ npm install
```

## Example usage
Go through the example in test/server.js

## Test

Run Unit Tests

```sh
$ gulp unit
```

Run Acceptance Tests

```sh
$ gulp acceptance
```

Run Code Coverage Tool

```sh
$ gulp cover
```
